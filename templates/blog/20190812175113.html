digを使ってMXレコードの値を確認する - CentOS

(CentOS5.7で確認) 　digコマンドを使ってMXレコードの値を確認するには以下のようにします。 ``` dig ドメイン名 mx ``` 　以下はサンプルです。 ``` dig perlcodesample.com mx ``` 　以下のような出力結果が返ってきます。 ``` ; <<>> DiG 9.3.6-P1-RedHat-9.3.6-4.P1.el5_4.2 <<>> perlcodesample.com mx ;; global options: printcmd ;; Got answer: ;; ->>HEADER<<- opcode: QUERY, status: NOERROR, id: 38789 ;; flags: qr rd ra; QUERY: 1, ANSWER: 1, AUTHORITY: 2, ADDITIONAL: 1 ;; QUESTION SECTION: ;perlcodesample.com. IN MX ;; ANSWER SECTION: perlcodesample.com. 3424 IN MX 10 mail.perlcodesample.com. ;; AUTHORITY SECTION: perlcodesample.com. 282 IN NS ns2.dns.ne.jp. perlcodesample.com. 282 IN NS ns1.dns.ne.jp. ;; ADDITIONAL SECTION: perlcodesample.com. 282 IN A 59.106.185.196 ;; Query time: 0 msec ;; SERVER: 192.168.5.2#53(192.168.5.2) ;; WHEN: Tue Jan 8 12:18:27 2013 ;; MSG SIZE rcvd: 113 ``` 　出力結果は多いですけれど、見る部分はANSWER SECTIONです。MXレコードが優先度10でmail.perlcodesample.com.に設定されているのが確認できます。
