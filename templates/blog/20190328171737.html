viエディタを使ってみる

　Windowsを使っていて初めてLinuxを使ったときに困るのは、文書の編集をどうやってしたらいいんだろうってことだと思います。LinuxではWebサーバーの設定やデータベースサーバーの設定を、設定ファイルを編集して行うことが普通です。 　なのでさくらVPSでサーバー管理を行うための最初の一歩としてはテキストファイルを編集する方法を覚える必要があります。 　でもコマンドラインの画面からどうやってテキストファイルを編集するのかなぁと疑問に思うかもしれませんが、コマンドラインで編集が可能なテキストエディタがLinuxにはあります。標準で仕えるviというエディタがあります。 ``` Linuxではテキストファイルを編集するのにviというエディタを使う ``` 　正直に延べるとviというエディタは、初めて使う人にとっては非常に使いにくいエディタです。Windowsのグラフィカルなエディタに慣れた人にとっては、Linuxを使いこなすための最初の難関となると思います。マウスが使えないので、すべての操作をキーボードで行う必要があるし、編集のモードという概念もあります。でもがんばる。 ##テキストファイルの作成 　テキストファイルを作成するには次のようにします。 ``` vi テキストファイル名 ``` 　viの引数にファイル名を渡すと、テキストが存在しない場合は新規に作成してくれます。foo.txtというファイルを作ってみましょう。 ``` vi foo.txt ``` 　viが立ち上がると思います。 ##テキストファイルの終了 　ここで初心者が困るのは、viエディタを終了する方法をわからないということだと思うので、これだけは一瞬で覚えてしまいましょう。以下の手順になります。 ``` まずEscを押す。 ``` 　Escを押すことでviはコマンドモードというモードに変更することができます。テキストエディタを終了するためには、まずEscを押して、コマンドモードにする必要があります。 ``` :q ``` 　:qと入力してEnterを押してください。viが終了します。qというエディタを終了させるコマンドを実行しています。コロンはコマンドの前に必要になります。 　またもう一つの保存しないで強制終了するためのコマンドも覚えておきましょう。viエディタでは、エディタの内容に変更がある場合は、qコマンドでは終了することができません。 　保存しないで強制終了するにはq!コマンドを使います。Escを押してコマンドモードに変更してから以下のように入力してEnterを押します。 ``` :q! ``` 　もうひとつ保存して終了も覚えておきましょう。wqとします。 ``` :wq ``` 　この三つの終了方法を覚えておけば、間違ってviを起動してしまっても、何とか大丈夫でしょう。 ``` qで終了 q!で保存しないで終了 wqで保存して終了 ``` 　では次回はviでテキストを編集する方法を紹介したいと思います。
