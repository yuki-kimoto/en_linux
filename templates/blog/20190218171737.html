Undo changes made to the index / Git

Undo changes made to the index You used the add command to update the index to commit, but you may want to undo the changes you made to the index. In such cases, use the reset command. 
git reset


This will undo the changes you made to the index. If you check it with the diff command, you can see that the index state is restored and the difference from the working tree is displayed.


git diff
