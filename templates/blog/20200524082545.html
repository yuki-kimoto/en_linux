To deploy Linux, deploy what is called a Linux distribution. Each Linux distribution is explained on each site.

  • Introduction to CentOS server construction

I want to experience Linux on a local machine on Windows

If you want to experience Linux on Windows, it's easy to use the tool msys2. You can build a pseudo Linux environment on Windows. In 2020, WSL2, a way to handle Linux on Windows, was introduced. You can build a real Linux environment on top of a lightweight virtual environment on Windows. You may try this.
