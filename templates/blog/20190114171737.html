CentOS5にLLVMとclangをインストールする | CentOS

　LLVMとclangはpapixさんの手順(http://hachiojipm.github.io/entry/2013-10-14-codegenerator-llvm.html)でインストールできるのですが、CentOS5では、前提としていくつか追加の手順が必要になります。CentOS6だと、pythonは2.6,gccは4.4なので、たぶん大丈夫だと思います。 ##llvmのインストールにはpython2.5以上が必要 　LLVMはインストールのときにpython2.5以上が必要です。epelというリポジトリを利用して、python 2.6をインストールしましょう。 　まずはCentOSのアーキテクチャの確認です。 ``` uname -a ``` 　x86_64が含まれていれば64bit版のCentOSです。i386が含まれていれば32bit版のCentOSです。 　64bit版のCentOSの場合は以下のコマンドでepelというRPMリポジトリを追加します。 ``` rpm -Uvh http://dl.fedoraproject.org/pub/epel/5/x86_64/epel-release-5-4.noarch.rpm ``` 　32bit版のCentOSの場合は以下のコマンドでepelというRPMリポジトリを追加します。 ``` rpm -Uvh http://dl.fedoraproject.org/pub/epel/5/i386/epel-release-5-4.noarch.rpm ``` 　パッケージが自動的に更新されない設定を行っておきます。下記のファイルを編集します。 ``` vi /etc/yum.repos.d/epel.repo ``` 　[epel]のセクションのenabledの値を0に設定しておきます ``` [epel] ... enabled=0 ``` 　python26パッケージをインストールします。 ``` yum -y install python26 --enablerepo=epel ``` 　python26がインストールされたかを確認します。 ``` python26 -V ``` 　バージョンなどが表示されればインストールに成功しています。 ##LLVMのインストールにはgcc 4.2以上が必要 　次にLLVMのインストールにはgcc 4.2以上が必要です。 　gcc 4.4をインストールします。 ``` yum -y install gcc44 ``` 　インストールされたかを確認します。 ``` gcc44 -v ``` 　C++をコンパイルするためのg++ 4.4もインストールします。 ``` yum -y install gcc44-c++ ``` 　インストールされたかを確認します。 ``` g++44 -v ``` ##LLVMとclangのインストール 　ここからやっとLLVMとclangをインストールします。 　以下の三つのファイルをダウンロードします。 ``` cd /usr/local/src wget http://llvm.org/releases/3.3/cfe-3.3.src.tar.gz wget http://llvm.org/releases/3.3/compiler-rt-3.3.src.tar.gz wget http://llvm.org/releases/3.3/llvm-3.3.src.tar.gz ``` 　展開します。 ``` tar -xf cfe-3.3.src.tar.gz tar -xf llvm-3.3.src.tar.gz tar -xf compiler-rt-3.3.src.tar.gz ``` 　llvm-3.3.srcのの中の適切な位置に、cfe-3.3.srcとcompiler-rt-3.3.srcをコピーします。 ``` cp -rp cfe-3.3.src llvm-3.3.src/tools/clang cp -rp compiler-rt-3.3.src llvm-3.3.src/projects/compiler-rt ``` 　llvm-3.3.srcディレクトリに移動して、設定を行います. configureを実行するときにgccとしてgcc44を、g++としてg++44を、pythonとしてpython26を使うようにします。 ``` cd llvm-3.3.src CC=gcc44 CXX=g++44 ./configure --with-python=/usr/bin/python26 ``` 　コンパイルします。1～2時間くらいはかかると思います。 ``` make ``` 　インストールします。 ``` make install ``` 　LLVMがインストールされたかどうかを確認します。 ``` llvm-config --version ``` 　バージョンが出力されればインストールが完了しています。
