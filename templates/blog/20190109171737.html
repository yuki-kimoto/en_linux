crontabの内容をバックアップする方法と読み込む方法 / CentOS

　crontab -eを使って内容を編集するのですが、直接編集するよりも、ファイルとして保存しておいて読み込むほうが確実ですね。特にバージョン管理システムで保存してある場合は、読み込むと楽です。 　またバックアップなどを取っておくと、すぐに修正できてよいですね。 ##crontabの内容をバックアップして保存 　-lオプションで内容を表示することができます。 ``` crontab -l > crontab.bak ``` ##crontabにファイルに内容を読み込む 　引数にファイルを指定するとcrontabに読み込むことができます。 ``` crontab ファイル名 ```
