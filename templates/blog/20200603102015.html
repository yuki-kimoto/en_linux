chownコマンド - 所有者ユーザーを変更する

chownコマンドを使うと、所有者ユーザーを変更することができます。 
chown 所有者ユーザー ファイル名
chownコマンドで、所有者ユーザーを変更するサンプルです。 
chown kimoto file

所有者グループも変更する

chownコマンドでは所有者ユーザーに加え所有者グループも変更することができます。 
chown 所有者ユーザー:所有者グループ ファイル名
所有者ユーザーに加え、所有者グループを変更するサンプルです。 
chown kimoto:myapp file

再帰的に所有者ユーザーを変更する

再帰的に所有者ユーザーを変更するには「-R」オプションを使います。 
chown -R kimoto dir
