gcc command --Compile and link C language source code

The gcc command is a command for compiling and linking C language source code. To compile and link and generate an executable file, use the "-o" option with the gcc command. 
gcc -o output file name input file name
Specify the output file name with the "-o" option. 
gcc -o output file name input file name
