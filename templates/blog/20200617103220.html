Setting environment variables for bash To set environment variables in bash, use the following environment variable setting syntax. 
export environment variable name = value


This is a sample to set environment variables. I set Perl's  PERL5LIB environment variable .


export PERL5LIB = / foo / lib



 What does export mean? 

If you use export, the environment variables will be inherited by the program to be executed.

If not used, environment variables are valid only within that shell.

Generally, environment variables are referenced by the program to be executed, so it is common to add export.
