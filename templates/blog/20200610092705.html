ln Command - Create Symbolic Link

Use the "-s" option of the ln command to create a symbolic link. 
ln -s Target LinkName
This is a sample to create a symbolic link of "/usr/bin" in the current directory. The main target is directories. 
ln -s /usr/bin mybin
ln is a command to create a link, and there are hard links in addition to symbolic links, but Web system developers and Linux server administrators mainly use symbolic links. Symbolic links are a type of file and can be deleted with the rm command.
